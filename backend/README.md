# Routes

## GET list of songs
GET /api/song

## GET specific song
GET /api/song/:id

## CREATE a song route
POST /api/song

## PUT route for EDITING specific song
PUT /api/song/:id

## DELETE specific song
DELETE /api/song/:id

## Attributes
title
artist
album
img_url
genre
key
mode
sales
streams
bpm
credits
structure (x8)
